# 恢复中

一个移动端优先的轻量 PWA，围绕畸胎瘤腹腔镜手术的术前准备、住院、术后恢复和两周休息来组织提醒、记录和放松内容。

## 功能

- 今日状态输入
- 自动整理今天只做三件事
- 今日提醒清单
- 身体记录
- 医生交代与需联系医院的情况
- 先歇一会儿
- 恢复期日记
- 本地存储
- PWA 基础配置

## 本地启动

```bash
pnpm install
pnpm dev
```

## 构建

```bash
pnpm build
pnpm preview
```

## 部署

### 最省事

- 直接部署到 `Vercel` 或 `Netlify`
- 构建命令用 `pnpm build`
- 输出目录是 `dist`
- 这是静态站点，通常不需要后端配置

### 如果部署到 GitHub Pages

- 这个仓库是项目页形式时，需要把站点基础路径改成仓库名对应的子路径
- 例如 `recovery-lite` 通常要使用类似 `/recovery-lite/` 的 base path
- 如果你要我继续帮你部署到 GitHub Pages，我可以直接帮你把这部分配置补齐

## 说明

- 数据默认保存在浏览器本地 `localStorage`
- 不含后端、账号系统或云同步
- `public/sw.js` 提供基础离线缓存
- `manifest.webmanifest` 支持添加到主屏幕

## 当前范围

- 默认围绕畸胎瘤腹腔镜这一条流程
- 默认本地保存，不接账号和后端
- 提醒、记录、日记都允许跳过

## 后续扩展方向

1. 把本地存储升级到 IndexedDB
2. 增加更细的提醒通知能力
3. 为云同步预留导入导出功能
